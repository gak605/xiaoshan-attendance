小山舍點名 PWA 安裝說明

1. 將此資料夾內所有檔案上傳到 GitHub Pages repository 根目錄。
2. GitHub repository → Settings → Pages。
3. Source 選 Deploy from a branch。
4. Branch 選 main，Folder 選 /(root)，按 Save。
5. 等 GitHub 產生網址後，用 iPhone Safari 開啟。
6. 點分享 → 加入主畫面。

欄位版本：
- 不含家長姓名
- 不含電話
- 學生資料只保留：姓名、單堂費用、星期、時段、備註、停課狀態

資料提醒：
目前資料存在瀏覽器 localStorage。請定期使用「匯出備份 JSON」或「匯出 CSV」。
